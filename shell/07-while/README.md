目标 : 
```
练习 while 语句的语法(主要用于密码验证)
解决在判断字符串相等的时候的一个小缺陷
之前的判断逻辑如下 : 
if [ $str1 == $str2 ] ; then
        # some code
if
这种方式有一个小问题就是当某一个字符串为空的时候 , 会报错
是因为假如第一个字符串为空 ("") , 第二个字符串为 "password", 那么整个 if 语句就会变成 : 
if [  == password ] ; then
        # some code
if
这样就出现了语法错误
因此 , 为了避免这样的问题 , 要养成良好的编码习惯 : 在字符串比较的时候在变量外包裹双引号
(单引号是不是更好呢 ? 不行 , 因为单引号只用于硬编码的比较)
if [ "$str1" == "$str2" ] ; then
        # some code
fi
```
